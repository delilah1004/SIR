회원 가입 방법 안내

회원 가입 방법

1. (https://delilah1004.github.io/sir-project-homepage/) 다음 주소로 접속합니다.

2. '데이터구축 프로그램 바로가기'를 클릭합니다.

3. 회원가입 버튼을 클릭합니다.

4. 아이디, 이메일, 이름 등 개인정보를 입력합니다.

5. 모두 동의를 클릭합니다.

6. 회원가입 버튼을 누릅니다.

프로젝트 참여 방법 안내

프로젝트 참여 방법

1. 아이디와 비밀번호를 입력하고 로그인을 합니다.

2. 라벨링 또는 검수 버튼을 클릭하여 프로젝트에 참여합니다.

3. 작업 할당받기 버튼을 클릭하여 작업을 시작합니다.

급여 지급 절차 안내

라벨러 급여 산정 방법

1. 작업 통계 메뉴에서 라벨링 통계를 확인합니다.

2. 이번 달 작업 완료 건수를 확인합니다.

3. 1건당 100원으로 산정합니다.

예) 위의 예시 이미지와 같이 작업 완료 건수가 10건이라면 10*100(원) 으로 계산하여 해당 월에는 총 1000원이 산정됩니다.

검수자 급여 산정 방법

* 작업통계에 표시된 작업 완료 건수로 계산하여 검수자는 1건당 50원으로 산정됩니다.

예) 작업통계에 표시된 작업 완료 건수가 6건이라면 6*50(원) 으로 총 300원이 산정됩니다.

라벨링 작업 안내

라벨링 작업 방법

1. 라벨링할 인물의 성별(Man or Woman)을 선택한다.

2. 선택한 태그에 해당하는 얼굴에 바운딩 박스를 그린다.

3. 바운딩 처리한 인물의 시선 방향을 체크한다. (바운딩 처리된 인물을 기준으로 시선의 방향 결정)

4. 바운딩 처리한 인물의 감정 상태를 체크한다.

검수 작업 안내

검수 작업 방법

1. 선택한 성별 태그와 바운딩 처리된 얼굴의 성별이 일치하는지 확인한다.

2. 바운딩 처리된 인물을 기준으로 해당 인물의 시선 방향이 올바르게 체크되었는지 확인한다.

3. 바운딩 처리된 인물을 기준으로 해당 인물의 시선 방향이 올바르게 체크되었는지 확인한다.

4. 발화의 어조를 통해 나타나는 화자의 감정 상태가 체크된 내용과 일치하는지 확인한다.

5. 발화의 내용을 통해 나타나는 화자의 발화 의도가 체크된 내용과 일치하는지 확인한다.

6. 자료 속 사람과 로봇의 상호작용 여부가 체크된 내용과 일치하는지 확인한다.
